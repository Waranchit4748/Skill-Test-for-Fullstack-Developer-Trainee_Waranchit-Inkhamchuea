# Expense Tracker Application

A comprehensive **Expense Tracker** application developed as part of a **Skill Test for Fullstack Developer Trainee** position. This application allows users to manage their daily expenses, categorize them, and visualize spending habits through an interactive dashboard.

**Live Demo:** https://skill-test-for-fullstack-developer.vercel.app/

## Features

- **Dashboard Overview:** View total spending, transaction count, and most used categories at a glance.
- **Data Visualization:** Interactive **Pie Chart** to visualize expense distribution by category.
- **Transaction Management:** Add, edit, and delete expense records.
- **Advanced Filtering:** Filter expenses by Date Range, Category, and Quick Filters (Today, 7 Days, 30 Days).
- **Sorting:** Sort transactions by Date (Newest/Oldest) or Amount (High/Low).
- **Responsive Design:** Fully responsive UI/UX for both Desktop and Mobile devices.
- **Modern UI:** Clean interface using **Tailwind CSS** and **Heroicons**.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Heroicons](https://heroicons.com/)
- **Database:** [MongoDB](https://www.mongodb.com/) (Atlas)
- **Date Handling:** date-fns
- **Fonts:** IBM Plex Sans Thai (Google Fonts)

## Getting Started Locally

1. **Clone the repository**
   git clone https://github.com/Waranchit4748/Skill-Test-for-Fullstack-Developer-Trainee_Waranchit-Inkhamchuea.git

2. **Install dependencies**
   npm install

3. **Set up Environment Variables** Create a .env.local file in the root directory and add your MongoDB connection string:
   MONGODB_URI=your_mongodb_connection_string_here

4. **Run the development server**
   npm run dev

5. Open your browser Navigate to http://localhost:3000 to see the application.
